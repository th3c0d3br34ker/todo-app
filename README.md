# Todo App

## Features ✨

✅ Insert New TASK.  
✅ Mark a TASK as complete.  
✅ Delete a TASK.

## Implementation 👨‍💻

✅ useState and useEffects Hooks.  
✅ localStorage for persistent Data Storage.
